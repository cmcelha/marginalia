# Public-domain translation checklist

A translation is its own copyrightable work — the original text being ancient does not make a modern translation public domain. Use this checklist before splitting any text.

## Safe rule of thumb

- **Translations first published before 1930 in the United States are public domain in the US.**
- **Crown Copyright (UK)** can make some pre-1930 British translations encumbered in the UK; for the King James Bible specifically, use the World English Bible if international distribution matters.
- If a translation's date is unclear or it was first published in the US between 1930 and 1977, **stop and verify**. Some are PD due to notice or renewal lapses; most aren't.

## Translations to refuse

If the user provides any of these, stop and ask for a PD substitute:

- **Walter Kaufmann's Nietzsche** (1950s-60s) — in copyright
- **Penguin Classics translations** post-1930 (Hays Meditations 2002, etc.) — in copyright
- **Victor Harris's Book of Five Rings** (1974), Wilson (2001), Cleary (1993) — all in copyright. No clean PD English translation of Five Rings exists; the text cannot currently be added to the library.
- **Foucault's Discipline and Punish** (1975) — in copyright. Bentham's 1791 *Panopticon Letters* is a PD substitute for the concept's origin.
- **Manufactured Consent** (Chomsky/Herman 1988) — in copyright, no PD substitute.

## How to verify when unsure

1. Check the title page for "first published" date.
2. Cross-reference Project Gutenberg — if Gutenberg hosts it, it's PD-cleared in the US.
3. For modern translations, check the copyright notice. Absence of notice on pre-1978 US publications can mean PD by lapse, but verify.
4. When in doubt, ask the user to confirm or pick a different translation.
